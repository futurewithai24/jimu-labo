# jimu-labo 新ツール追加手順

最終更新：2026-07-10

---

## 全体の流れ

```
① pagesフォルダに新しい .py ファイルを作る
② ローカルで動作確認する（任意）
③ requirements.txt を更新する（新ライブラリを使う場合のみ）
④ git add → commit → push
⑤ Streamlit Cloudで反映確認する
```

---

## ① pagesフォルダにファイルを作る

**場所：**
```
04_開発部\Streamlit\pages\
```

**命名ルール：**
```
（番号）_（ツール名）.py
例：9_画像透かし追加.py
    10_フォルダ自動作成.py
```

番号はサイドバーの表示順になります。一番大きい番号の次をつけてください。

## ② コードの基本テンプレート

新しいページはこのテンプレートをコピーして作ります。

```python

st.page_link("home.py", label="← ホームに戻る")
st.title("🔧 ツール名")
st.caption("このツールの説明を1行で")

# ── サイドバー（設定）──────────────────────────
with st.sidebar:
    st.header("設定")
    # ここに設定項目を追加する
    # 例：
    # value = st.text_input("入力してください", value="初期値")

# ── ファイルアップロード ──────────────────────

    "ファイルをアップロード",
    type=["xlsx", "csv"]  # 対応形式を変える
)

if uploaded_file:
    # ── 処理 ────────────────────────────────────
    if st.button("実行する", type="primary"):
        with st.spinner("処理中です…"):
            pass  # ← ここに処理を書く

        st.success("完成しました！")

        # ── ダウンロード ──────────────────────
        # st.download_button(
        #     label="ダウンロード",
        #     data=結果データ,
        #     file_name="output.xlsx",
        #     mime="application/vnd.openxmlformats-officedocument.spreadsheetml.sheet"
        # )

# ── 使い方 ──────────────────────────────────────
with st.expander("📖 使い方"):
    st.write("1. ファイルをアップロードする")
    st.write("2. 設定を確認する")
    st.write("3. 「実行する」ボタンを押す")
```

## ④ GitHubへpushする

PowerShellまたはコマンドプロンプトで以下を実行します。

```powershell
# Streamlitフォルダに移動
cd "E:\Maruno-Mai-AI-Office\04_開発部\Streamlit"

# 変更を確認
git status

# 追加したファイルをステージング
git add pages/9_画像透かし追加.py

# requirements.txtも変えた場合
git add requirements.txt

# コミット
git commit -m "〇〇ツールを追加"

# push（Streamlit Cloudに自動反映される）
git push
```

---

## ⑤ Streamlit Cloudで確認する

1. https://jimu-labo.streamlit.app/ を開く
2. サイドバーに新しいツールが追加されているか確認
3. 反映されない場合は1〜2分待ってリロードする

それでも反映されない場合：
- Streamlit Cloudにログインする
- `Manage app` → `Reboot app` を押す

## よく使うコードパターン

### 複数ファイルのアップロード
```python
files = st.file_uploader("画像を選択", type=["jpg", "png"], accept_multiple_files=True)
```

### ZIPでダウンロード
```python
import zipfile, io

zip_buf = io.BytesIO()
with zipfile.ZipFile(zip_buf, "w") as zf:
    zf.writestr("ファイル名.txt", "内容")
zip_buf.seek(0)

st.download_button("ZIPをダウンロード", data=zip_buf, file_name="output.zip", mime="application/zip")
```

### Excelをダウンロード
```python
import io, pandas as pd

buf = io.BytesIO()
df.to_excel(buf, index=False)
buf.seek(0)

st.download_button("Excelをダウンロード", data=buf, file_name="output.xlsx",
                   mime="application/vnd.openxmlformats-officedocument.spreadsheetml.sheet")
```

## 注意事項

- ファイル名に日本語を使ってもOK（GitHubはエンコードして管理する）
- APIキーやパスワードはコードに直書きしない（`.streamlit/secrets.toml` を使う）
- Excelのフォントは必ず **Meiryo UI** に統一（openpyxl使用時）
- 元ファイルを上書きする処理は書かない（必ず別ファイルに出力する）
