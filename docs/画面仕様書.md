```python
import streamlit as st

    page_title="じむラボ｜業務効率化ツール集",

    layout="wide"
)

  <h1 style='color: #1E3A5F; font-size: 2.2rem;'>ファイル作業、もうラクにしよう。</h1>
  <p style='color: #555; font-size: 1.1rem;'>
    CSVをアップして、ボタンを押すだけ。<br>
    面倒な事務作業を自動化するツール集です。
  </p>
</div>
""", unsafe_allow_html=True)

st.divider()

# ── こんな人に使ってほしい ──
st.subheader("こんな人に使ってほしい")
col1, col2, col3 = st.columns(3)
with col1:
    st.info("📅 Excelの作業に毎回時間がかかっている")
with col2:
    st.info("📄 PDFの結合・分割を手作業でやっている")
with col3:
    st.info("📈 グラフを毎月ゼロから作り直している")

# ── ツール一覧 ──
st.subheader("ツール一覧")
col1, col2 = st.columns(2)

with col1:
    with st.container(border=True):
        st.markdown("### 📊 ガントチャート自動生成")
        st.write("タスク一覧のCSVをアップするだけで、Excel形式のガントチャートを自動作成。日単位・週単位に対応。")
        st.page_link("pages/1_ガントチャート.py", label="使ってみる →", icon="📊")
```

## 10. CSSカスタマイズ一覧（app.py）

`app.py` の `<style>` ブロックで管理。変更はすべてここに集約されている。

```css
h1 { font-size: 28px; }          /* アプリ名（じむラボ） */
h3 { font-size: 16px; }          /* ツール一覧の名前 */
h4 { font-size: 14px; }          /* 使い方3ステップの見出し */
p  { font-size: 14px; }          /* 本文テキスト */

h4 a, h3 a, h2 a { display: none; }          /* 見出しのリンクアイコン非表示 */
[data-testid="stPageLink"] * { font-weight: bold; }  /* 「使ってみる→」Bold */

.stApp {
    background:                   /* 宇宙系ブルー背景（ダークモード対応） */
        radial-gradient(ellipse at top left,     rgba(0, 100, 255, 0.45) ...),
        radial-gradient(ellipse at top right,    rgba(0, 200, 255, 0.25) ...),
        radial-gradient(ellipse at bottom left,  rgba(60, 0, 200, 0.35)  ...),
        radial-gradient(ellipse at bottom right, rgba(0, 140, 255, 0.40) ...);
}
```

> 背景色の濃さは各 `rgba()` の第4引数（0〜1）で調整できる。
